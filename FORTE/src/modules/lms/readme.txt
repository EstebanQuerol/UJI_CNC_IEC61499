This directory contains FBs for accesing the I/Os of the Lego Mindstorms nxt controller:

 LMS_USB_SRV ... communication function block using the serial protocol over USB to provide the management communication with the tool
 LMS_DEV
 LMS_RMT_RES
 LMS_LED ... set the led of the light sensor
